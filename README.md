## Git LFS

Large binary files are tracked via Git LFS. Tracked types: `.psd`, `.psb`, `.tga`, `.tiff`, `.exr`, `.fbx`, `.blend`, `.wav`, `.tif`, `.mp3`, `.ogg`, `.mp4`, `.mov`, `.unitypackage`.

After cloning, run:
```bash
git lfs install
git lfs pull
```
